# Cybersecurity 101 — runtime dependencies
#
# Python 3.10+ required (uses PEP 604 `X | Y` annotations). Tested on 3.12.
#
# 27 of the 36 scripts run on the standard library alone. The other 9 use an
# optional third-party package and degrade gracefully when it is missing — they
# print a "... skipped (missing dependency: X)" line and exit 0 — EXCEPT
# `rsa_encrypt_decrypt.py`, which imports `cryptography` at module level and
# therefore needs it installed to run.
#
# Install everything:    pip install -r requirements.txt
# Install selectively:   pip install pycryptodome cryptography
